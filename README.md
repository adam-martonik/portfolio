# Osobné portfólio | Adam Martonik

Zdrojový kód môjho osobného portfólia, kde prezentujem svoje programátorské zručnosti, univerzitné/osobné projekty a pracovné skúsenosti. Stránka je navrhnutá ako responzívna Single Page Application (SPA) s moderným "dark mode" dizajnom.

🌐 **[Pozrieť live verziu webu](https://adam-martonik.github.io/portfolio)**

## 🚀 Funkcionality

- **Moderný dizajn:** Tmavá téma s výraznými modrými akcentmi prispôsobená pre vývojárov.
- **Plynulé scrollovanie:** Plne funkčné navigačné menu s plynulým prechodom medzi sekciami.
- **Oddelenie dát od logiky:** Všetky osobné informácie, zručnosti a projekty sú uložené v samostatnom konfiguračnom súbore `data.js` pre jednoduchú a rýchlu aktualizáciu.
- **Rýchlosť:** Využitie Vite pre bleskový lokálny vývoj a optimalizovaný produkčný build.

## 🛠️ Technologický stack

- **Frontend:** React (JavaScript)
- **Build tool:** Vite
- **Štýlovanie:** Čisté CSS (Custom vlastnosti a Flexbox/Grid systém)
- **Deployment:** GitHub Pages (`gh-pages`)
